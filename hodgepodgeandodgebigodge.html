TEMPLATE DASH APP

TEMPLATE DASH APP

This is a simple template dash app, designed to be easily editable for new Dash users. All it does is plot an exponential function, where the exponent is variable with a slider. It's to demonstrate a code sectionalization idea I use for larger apps. The code can be accessed on Github and the app is deployed on Heroku (free version so it takes ~10s to load).

THE SPREAD OF COVID-19

THE SPREAD OF COVID-19

In April 2020 I wrote a program to visualize the spread of COVID 19. I wrote this in Matlab, which ultimately meant that I had to post static videos, rather than allowing users to play with the model themselves. (This was the catalyst for me learning Dash.) It's still not user-friendly, and I dropped it before comparing to real-world scenarios, but it was a fun project.
